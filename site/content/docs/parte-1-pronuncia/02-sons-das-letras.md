{{< callout type="info" >}}
Os sons aprendem-se melhor ouvindo-os pronunciar corretamente. Esta seção serve mais para
consulta do que para ser decorada. As comparações abaixo foram **adaptadas ao português
brasileiro**.
{{< /callout >}}

**§ 4.** O latim não era pronunciado como o português. Os romanos, no início da era cristã,
pronunciavam sua língua substancialmente como se descreve a seguir.

## As vogais

**§ 5.** As vogais têm os seguintes sons (¯ marca a vogal longa; ˘, a breve):

| Vogal | Som (equivalente em português) | Exemplos latinos |
|:---:|---|---|
| **ā** | como o *a* de "p**á**", porém prolongado | {{< lat "hāc, stās" >}} |
| **ă** | como o *a* de "c**a**sa", breve | {{< lat "ă´-măt, că-nās" >}} |
| **ē** | como o *ê* fechado de "m**ê**s" | {{< lat "tē´-lă, mē´-tă" >}} |
| **ĕ** | como o *é* aberto de "p**é**" | {{< lat "tĕ´-nĕt, mĕr´-cēs" >}} |
| **ī** | como o *i* de "v**i**da", prolongado | {{< lat "sĕr´-tī, prā´-tī" >}} |
| **ĭ** | como o *i* de "f**i**lho", breve | {{< lat "sĭ´-tĭs, bĭ´-bī" >}} |
| **ō** | como o *ô* fechado de "av**ô**" | {{< lat "Rō´-mă, ō´-rĭs" >}} |
| **ŏ** | como o *ó* aberto de "p**ó**" | {{< lat "mŏ´-dŏ, bŏ´-nōs" >}} |
| **ū** | como o *u* de "**u**va", prolongado | {{< lat "ū´-mŏr, tū´-bĕr" >}} |
| **ŭ** | como o *u* de "per**u**", breve | {{< lat "ŭt, tū´-tŭs" >}} |

{{< callout emoji="✏️" >}}
**Observação.** Há diferença nítida de som entre a vogal longa e a breve (salvo no caso do
*a*): não é apenas questão de *duração*, mas também de *timbre* — daí, em português, a oposição
entre as vogais fechadas (ê, ô) e abertas (é, ó).
{{< /callout >}}

## Os ditongos

**§ 6.** Nos **ditongos** (sons de duas vogais), ambas as vogais são ouvidas numa única
sílaba.

| Ditongo | Som (equivalente em português) | Exemplos latinos |
|:---:|---|---|
| **ae** | como "ai" em "p**ai**" | {{< lat "tae´-dae" >}} |
| **au** | como "au" em "P**au**lo" | {{< lat "gau´-dĕt" >}} |
| **ei** | como "ei" em "r**ei**" | {{< lat "dein´-dĕ" >}} |
| **eu** | como "eu" em "c**éu**" (e breve + u, numa sílaba) | {{< lat "seu" >}} |
| **oe** | como "ói" em "her**ói**" | {{< lat "foe´-dŭs" >}} |
| **ui** | como "ui" em "f**ui**" (u + i, numa sílaba) | {{< lat "cui, huic" >}} |

Dê a cada vogal e ditongo o seu som próprio, sem atropelá-los nas sílabas átonas.

## As consoantes

**§ 7.** As **consoantes** pronunciam-se como em português, exceto:

| Consoante | Som (equivalente em português) | Exemplos latinos |
|---|---|---|
| **c** | sempre /k/, como em "**c**asa"; nunca como em "**c**ebola" | {{< lat "că´-dō, cĭ´-bŭs, cē´-nă" >}} |
| **g** | sempre /g/, como em "**g**ato"; nunca como em "**g**elo" | {{< lat "gĕ´-mō, gĭg´-nō" >}} |
| **i consoante** | como o *i* de "**i**ate" (/j/, como em "ma**i**o") | {{< lat "iăm, iŏ´-cŭs" >}} |
| **n** antes de *c, qu, g* | nasal velar /ŋ/, como em "ta**n**go", "ba**n**co" | {{< lat "ăn´-cŏ-ră (âng-co-ra)" >}} |
| **qu, gu** (e às vezes **su**) antes de vogal | /kw/, /gw/, /sw/, como em "**qu**ase", "á**gu**a", "**su**ave"; o *u* é semivogal | {{< lat "ĭn´-quĭt, lĭn´-guă, suā´-dĕ-ō" >}} |
| **s** | sempre surdo /s/, como em "**s**apo"; nunca /z/, como em "ca**s**a" | {{< lat "rŏ´-să, ĭs" >}} |
| **t** | sempre /t/, como em "**t**omate"; nunca "tch" — {{< lat "ratiō" >}} = "rá-ti-o", não "rá-tchi-o" | {{< lat "ră´-tĭ-ō, nā´-tĭ-ō" >}} |
| **v** | como o *u* semivogal /w/ (o "u" de "**qu**ase"); *nunca* como o *v* de "vinho" ({{< lat "vīnum" >}} ≈ "uí-num") | {{< lat "vī´-nŭm, vĭr" >}} |
| **x** | vale por duas consoantes (cs/gs): /ks/, como em "tá**x**i", "fi**x**o" | {{< lat "ĕx´-trā, ĕx-āc´-tŭs" >}} |
| **bs, bt** | soam "ps" e "pt" ({{< lat "urbs" >}} ≈ "urps") | {{< lat "ŭrbs, ŏb-tĭ´-nĕ-ō" >}} |
| **ch, ph, th** | soam /k/, /p/, /t/ ({{< lat "pulcher" >}} ≈ "púl-ker") | {{< lat "pŭl´-chĕr, Phoe´-bē, thĕ-ā´-trŭm" >}} |

{{< callout emoji="✏️" >}}
*a.* Nos grupos de consoantes, dê a cada uma o seu som distinto. As consoantes **dobradas**
devem ser pronunciadas com uma breve pausa entre os dois sons — algo que o português não tem
por natureza. Pronuncie como se houvesse uma divisão: {{< lat "mit-tō" >}},
{{< lat "Ap-pius" >}}, {{< lat "bel-lum" >}}.
{{< /callout >}}
