# Lição X — Segunda Declinação (continuação)

> Tradução-base fiel de *Latin for Beginners* (D'Ooge), §§ 79–82.

## § 79 — Adjetivos Masculinos e Neutros da Segunda Declinação

Temos usado livremente adjetivos femininos, como **bona**, concordando com substantivos femininos da primeira declinação e declinados como eles. Os adjetivos *masculinos* dessa classe declinam-se como **dominus**, e os *neutros*, como **pīlum**. O adjetivo e o substantivo, masculino e neutro, declinam-se, portanto, da seguinte forma:

| | Substantivo e adjetivo masculinos | Term. | Substantivo e adjetivo neutros | Term. |
|--|----------------------------------|-------|-------------------------------|-------|
| | **dominus bonus**, *o bom senhor* | | **pīlum bonum**, *a boa lança* | |
| | Radicais **domin- bon-** | | Radicais **pīl- bon-** | |
| | **Singular** | | | |
| *Nom.* | do´min**us** bon**us** | **-us** | pīl**um** bon**um** | **-um** |
| *Gen.* | domin**ī** bon**ī** | **-ī** | pīl**ī** bon**ī** | **-ī** |
| *Dat.* | domin**ō** bon**ō** | **-ō** | pīl**ō** bon**ō** | **-ō** |
| *Ac.* | domin**um** bon**um** | **-um** | pīl**um** bon**um** | **-um** |
| *Abl.* | domin**ō** bon**ō** | **-ō** | pīl**ō** bon**ō** | **-ō** |
| *Voc.* | domin**e** bon**e** | **-e** | pīl**um** bon**um** | **-um** |
| | **Plural** | | | |
| *Nom.* | domin**ī** bon**ī** | **-ī** | pīl**a** bon**a** | **-a** |
| *Gen.* | domin**ō´rum** bon**ō´rum** | **-ōrum** | pīl**ō´rum** bon**ō´rum** | **-ōrum** |
| *Dat.* | domin**īs** bon**īs** | **-īs** | pīl**īs** bon**īs** | **-īs** |
| *Ac.* | domin**ōs** bon**ōs** | **-ōs** | pīl**a** bon**a** | **-a** |
| *Abl.* | domin**īs** bon**īs** | **-īs** | pīl**īs** bon**īs** | **-īs** |

Decline juntos: **bellum longum, equus parvus, servus malus, mūrus altus, frūmentum novum**.

## § 80 — Aposto

Observe as frases:

| Frase | Tradução |
|-------|----------|
| **Lesbia ancilla est bona** | *Lesbia, a criada, é boa* |
| **Fīlia Lesbiae ancillae est bona** | *a filha de Lesbia, a criada, é boa* |
| **Servus Lesbiam ancillam amat** | *o escravo ama Lesbia, a criada* |

Nessas frases, **ancilla**, **ancillae** e **ancillam** indicam a classe de pessoas à qual *Lesbia* pertence e explicam quem ela é. Substantivos assim relacionados — de modo que o segundo é apenas outro nome para o primeiro e o explica — dizem-se em *aposto*, e estão sempre no mesmo caso.

## § 81 — Regra: Aposto

**Regra.** *O aposto concorda em caso com o substantivo que explica.*

## § 82 — Exercícios

Estude antes o vocabulário especial, p. 285.

**I.**

1. Patria servī bonī, vīcus servōrum bonōrum, bone popule. 2. Populus oppidī magnī, in oppidō magnō, in oppidīs magnīs. 3. Cum pīlīs longīs, ad pīla longa, ad mūrōs lātōs. 4. Lēgāte male, amīcī legātī malī, cēna grāta dominō bonō. 5. Frūmentum equōrum parvōrum, domine bone, ad lēgātōs clārōs. 6. Rhēnus est in Germāniā, patriā meā. 7. Sextus lēgātus pīlum longum portat. 8. Oppidānī bonī Sextō lēgātō clārā pecūniam dant. 9. Malī servī equum bonum Mārcī dominī necant. 10. Galba agricola et Iūlia fīlia bona labōrant. 11. Mārcus nauta in īnsulā Siciliā habitat.

**II.** *(Exercício originalmente em inglês → latim; convertido para português → latim.)*

1. Escravo mau, quem é seu amigo? Por que ele não elogia Galba, seu senhor? 2. Meu amigo é de (**ex**) uma aldeia da Germânia, minha pátria. 3. Meu amigo não ama o povo da Itália. 4. Quem está cuidando¹ do bom cavalo de Galba, o lavrador? 5. Marco, onde está Lesbia, a criada? 6. Ela está indo depressa¹ para a casinha² de Júlia, filha do lavrador.

> ¹ Ver nota 1, p. 33. Lembre-se de que **cūrat** é transitivo e rege um objeto direto.
