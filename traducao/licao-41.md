# Lição XLI — Terceira Declinação · Radicais Consonantais Neutros *(Conclusão)*

> Tradução-base fiel de *Latin for Beginners* (D'Ooge), §§ 238–239.

## § 238 — Neutros Consonantais

Os radicais consonantais neutros não acrescentam terminação alguma no nominativo e se declinam da seguinte forma:

| | **flūmen**, n. *rio* | **tempus**, n. *tempo* | **opus**, n. *obra* | **caput**, n. *cabeça* | Term. |
|---|---|---|---|---|---|
| **Bases** | **flūmin-** | **tempor-** | **oper-** | **capit-** | |
| | **Singular** | | | | |
| *Nom.* | flūmen | tempus | opus | caput | — |
| *Gen.* | flūmin**is** | tempor**is** | oper**is** | capit**is** | **-is** |
| *Dat.* | flūmin**ī** | tempor**ī** | oper**ī** | capit**ī** | **-ī** |
| *Acc.* | flūmen | tempus | opus | caput | — |
| *Abl.* | flūmin**e** | tempor**e** | oper**e** | capit**e** | **-e** |
| | **Plural** | | | | |
| *Nom.* | flūmin**a** | tempor**a** | oper**a** | capit**a** | **-a** |
| *Gen.* | flūmin**um** | tempor**um** | oper**um** | capit**um** | **-um** |
| *Dat.* | flūmin**ibus** | tempor**ibus** | oper**ibus** | capit**ibus** | **-ibus** |
| *Acc.* | flūmin**a** | tempor**a** | oper**a** | capit**a** | **-a** |
| *Abl.* | flūmin**ibus** | tempor**ibus** | oper**ibus** | capit**ibus** | **-ibus** |

1. Revise § 74 e aplique as regras a esta declinação.

2. Bases em **-in-** têm **-e-** no lugar de **-i-** no nominativo, como **flūmen**, base **flūmin-**.

3. A maioria das bases em **-er-** e **-or-** têm **-us** no nominativo, como **opus**, base **oper-**; **tempus**, base **tempor-**.

## § 239 — Exercícios

Aprenda primeiro o vocabulário especial, p. 292.

**I.** 1. Barbarī ubi Rōmam cēpērunt, maxima rēgum opera dēlēvērunt. 2. Rōmānī multās calamitātēs ā barbarīs accēpērunt. 3. Ubi erat summus terror apud oppidānōs, animī dubiī eōrum ab ōrātōre clārō cōnfīrmātī sunt. 4. Rōma est in rīpīs flūminis magnī. 5. Ubi Caesar imperātor mīlitēs suōs arma capere iussit, iī ā proeliō continērī nōn potuērunt. 6. Ubi proelium factum est, imperātor reperīrī nōn potuit. 7. Imperātor sagittā in capite vulnerātus erat et stāre nōn poterat. 8. Eum magnō labōre pedes ex proeliō portāvit. 9. Is bracchiīs suīs imperātōrem tenuit et eum ex perīculīs summīs servāvit. 10. Virtūte suā bonus mīles ab imperātōre corōnam accēpit.

**II.** *(Traduza para o latim — exercício originalmente em inglês)*

1. O cônsul colocou uma coroa na cabeça do vencedor.
2. Diante dos portões foi recebido pelos habitantes.
3. Um famoso orador o elogiou e disse: "Com os teus trabalhos salvaste a pátria do desastre."
4. As palavras do orador foram agradáveis ao vencedor.
5. Salvar a pátria era uma grande tarefa.
