# Lição I — Primeiros Princípios

> Tradução-base fiel de *Latin for Beginners* (D'Ooge), §§ 19–22.

## § 19 — Sujeito e Predicado

O latim, assim como o português, expressa pensamentos por meio de orações. Uma oração é uma combinação de palavras que expressa um pensamento e, em sua forma mais simples, é a afirmação de um único fato. Por exemplo:

| | |
|---|---|
| *Galba é um lavrador* | *O marinheiro combate* |
| **Galba est agricola** | **Nauta pugnat** |

Em cada uma dessas orações há duas partes:

| Sujeito | | Predicado | |
|---|---|---|---|
| *Galba* | **Galba** | *é um lavrador* | **est agricola** |
| *O marinheiro* | **Nauta** | *combate* | **pugnat** |

2. O sujeito é a pessoa, lugar ou coisa sobre a qual algo é dito, sendo, portanto, um *substantivo* ou alguma palavra que possa cumprir essa função.

> *a.* Os pronomes, como o próprio nome indica (*pro*, "em vez de", e *nome*), frequentemente tomam o lugar dos substantivos, geralmente para evitar a repetição do mesmo substantivo: *Galba é um lavrador; **ele** é um homem robusto.*

3. O predicado é aquilo que se diz sobre o sujeito, e consiste em um verbo com ou sem modificadores.

> *a.* Um verbo é uma palavra que afirma algo (geralmente uma ação) acerca de uma pessoa, lugar ou coisa.

---

## § 20 — O Objeto

Nas duas orações *O menino acertou a bola* e *A bola acertou o menino*, as mesmas palavras são usadas, mas o sentido é diferente e depende da ordem das palavras. O **agente** da ação — aquilo sobre o qual algo é dito — é, como vimos acima, o **sujeito**. **Aquilo sobre o qual a ação recai** é o **objeto direto** do verbo. A oração *O menino acertou a bola* é, portanto, analisada da seguinte forma:

| Sujeito | Predicado |
|:---:|:---:|
| *O menino* | *acertou a bola* |
| | (verbo) (objeto direto) |

> *a.* Um verbo cuja ação recai diretamente sobre o objeto, como na oração acima, é chamado de **verbo transitivo**. Um verbo que não admite objeto direto é chamado de **intransitivo**, como em *eu caminho, ele vem*.

---

## § 21 — A Cópula

O verbo *ser* em suas diferentes formas — *és*, *é*, *era*, *foi*, etc. — não nos diz nada acerca do sujeito; tampouco rege um objeto. Ele simplesmente conecta o sujeito à palavra ou às palavras do predicado que possuem sentido próprio. Por isso, é chamado de **cópula**, isto é, *o elo* ou *o conector*.

## § 22 — Exercício de Identificação

Pronuncie as orações latinas a seguir e identifique os *substantivos, verbos, sujeitos, objetos, predicados* e *cópulas*:

| | |
|---|---|
| 1. | **America est patria mea** |
| | *América é minha pátria* |
| 2. | **Agricola fīliam amat** |
| | *O lavrador ama (sua) filha* |
| 3. | **Fīlia est Iūlia** |
| | *(Sua) filha é Júlia* |
| 4. | **Iūlia et agricola sunt in īnsulā** |
| | *Júlia e o lavrador estão na ilha* |
| 5. | **Iūlia aquam portat** |
| | *Júlia carrega água* |
| 6. | **Rosam in comīs habet** |
| | *Tem uma rosa no (seu) cabelo* |
| 7. | **Iūlia est puella pulchra** |
| | *Júlia é uma menina bonita* |
| 8. | **Domina fīliam pulchram habet** |
| | *A senhora tem uma filha bonita* |

> *a.* As orações acima mostram que o latim não expressa algumas palavras que são necessárias no português. Em primeiro lugar, *o latim não tem artigo — nem definido nem indefinido*: **agricola**, por exemplo, pode significar *o lavrador*, *um lavrador* ou simplesmente *lavrador*.

> ⚠️ **[revisar — comparação com inglês]** O original diz que o latim não expressa pronomes pessoais (*I, you, he, she*) e pronomes possessivos (*my, your, his, her*), contrastando com o inglês. Em português, também omitimos os pronomes sujeito quando o sentido é claro pelo contexto (diz-se *luto*, não necessariamente *eu luto*), e os pronomes possessivos igualmente costumam ser omitidos. Portanto, o contraste com o português é menos acentuado do que com o inglês — a nota poderia ser reformulada como: "além disso, o latim omite os pronomes pessoais e possessivos quando o sentido da oração os dispensa, o que também ocorre em português."
