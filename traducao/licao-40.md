# Lição XL — Terceira Declinação · Radicais Consonantais *(Continuação)*

> Tradução-base fiel de *Latin for Beginners* (D'Ooge), §§ 235–237.

## § 235 — Classe II

Os radicais consonantais que não acrescentam terminação alguma no nominativo declinam-se nos outros casos exatamente como os da Classe I. Podem ser masculinos, femininos ou neutros.

## § 236 — Paradigmas

**Masculinos e Femininos**

| | **cōnsul**, m. *cônsul* | **legiō**, f. *legião* | **ōrdō**, m. *fileira* | **pater**, m. *pai* | Term. M./F. |
|---|---|---|---|---|---|
| **Bases** | **cōnsul-** | **legiōn-** | **ōrdin-** | **patr-** | |
| | **Singular** | | | | |
| *Nom.* | cōnsul | legiō | ōrdō | pater | — |
| *Gen.* | cōnsul**is** | legiōn**is** | ōrdin**is** | patr**is** | **-is** |
| *Dat.* | cōnsul**ī** | legiōn**ī** | ōrdin**ī** | patr**ī** | **-ī** |
| *Acc.* | cōnsul**em** | legiōn**em** | ōrdin**em** | patr**em** | **-em** |
| *Abl.* | cōnsul**e** | legiōn**e** | ōrdin**e** | patr**e** | **-e** |
| | **Plural** | | | | |
| *Nom.* | cōnsul**ēs** | legiōn**ēs** | ōrdin**ēs** | patr**ēs** | **-ēs** |
| *Gen.* | cōnsul**um** | legiōn**um** | ōrdin**um** | patr**um** | **-um** |
| *Dat.* | cōnsul**ibus** | legiōn**ibus** | ōrdin**ibus** | patr**ibus** | **-ibus** |
| *Acc.* | cōnsul**ēs** | legiōn**ēs** | ōrdin**ēs** | patr**ēs** | **-ēs** |
| *Abl.* | cōnsul**ibus** | legiōn**ibus** | ōrdin**ibus** | patr**ibus** | **-ibus** |

1. Com exceção do nominativo, as terminações são exatamente as mesmas da Classe I, e a base ou radical se encontra da mesma forma.

2. Masculinos e femininos com bases em **-in-** e **-ōn-** perdem o **-n-** e terminam em **-ō** no nominativo, como **legiō** (base **legiōn-**), **ōrdō** (base **ōrdin-**).

3. Bases em **-tr-** têm **-ter** no nominativo, como **pater** (base **patr-**).

4. Note como o genitivo singular fornece a chave para toda a declinação. *Aprenda-o sempre com o nominativo.*

## § 237 — Exercícios

Aprenda primeiro o vocabulário especial, p. 291.

**I.** 1. Audīsne tubās, Mārce? Nōn sōlum tubās audiō sed etiam ōrdinēs mīlitum et carrōs impedīmentōrum plēnōs vidēre possum. 2. Quās legiōnēs vidēmus? Eae legiōnēs nūper ex Galliā vēnērunt. 3. Quid ibi fēcērunt? Studēbantne pugnāre an sine virtūte erant? 4. Multa proelia fēcērunt[^1] et magnās victōriās et multōs captīvōs reportāvērunt. 5. Quis est imperātor eārum legiōnum? Caesar, summus Rōmānōrum imperātor. 6. Quis est eques quī pulchram corōnam gerit? Is eques est frāter meus. Eī corōna ā cōnsule data est quia summā virtūte pugnāverat et ā barbarīs patriam servāverat.

[^1]: **proelium facere** = *travar uma batalha*.

**II.** *(Traduza para o latim — exercício originalmente em inglês)*

1. Quem viu meu pai hoje?
2. Eu o vi há pouco (**nūper**). Estava se apressando para a tua casa com tua mãe e irmã.
3. Quando os homens estão longe da pátria e carecem de alimento, não podem ser impedidos[^2] de praticar o mal[^3].
4. A segurança dos soldados é cara a César, o general.
5. Os chefes estavam ansiosos para atacar uma cidade repleta de trigo, que estava nas mãos do cônsul.
6. O rei proibiu que os pertences dos cativos fossem destruídos.

[^2]: **contineō**. Cf. § 180.
[^3]: Abl. **iniūriā**.
