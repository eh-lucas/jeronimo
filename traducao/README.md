# Tradução-base do D'Ooge (PT-BR)

Esta pasta guarda a **tradução fiel** de *Latin for Beginners* (B. L. D'Ooge, 1911) para o
português brasileiro — a **espinha teórica** do projeto Jerônimo. É material de **referência**,
usado ao longo do curso; **não** é ainda a versão adaptada à Vulgata (essa adaptação —
exemplos bíblicos, leituras, pronúncia eclesiástica, exercícios reorientados — acontece nas
fases seguintes, ver [`../PLANO-JERONIMO.md`](../PLANO-JERONIMO.md)).

## Convenções

- Um arquivo Markdown por lição: `licao-01.md` … `licao-30.md`.
- Tradução fiel ao D'Ooge, seguindo [`../REGRAS-TRADUCAO.md`](../REGRAS-TRADUCAO.md).
- Latim preservado intacto (com macrons ā / breves ă); exemplos do próprio D'Ooge mantidos.
- Pontos delicados de comparação com o inglês, que normalmente seriam perguntados ao usuário,
  ficam marcados no texto com:
  `> ⚠️ **[revisar — comparação com inglês]** …`
  para revisão humana posterior.

## Conteúdo

- `licao-01.md` … `licao-30.md` — Lições I a XXX (em produção por agentes).
- `manual.html` — versão HTML antiga (intro + Parte I + Lições I–II), anterior ao site Hugo;
  mantida por ora, será aposentada.
